# AI Prompts Used

## AI Tool
ChatGPT (OpenAI)

### Prompt 1
Act as my coach and guide me step by step while building a DevConf 2026 conference website using HTML and CSS. Help me learn instead of completing the project for me.

### Prompt 2
Review my HTML and CSS code, identify mistakes, explain the issues, and suggest how I can fix them.

### Prompt 3
Suggest improvements for the website layout, spacing, typography, colors, and overall user experience without changing my original project idea.

### Prompt 4
Provide hints and explanations for implementing different sections such as the Hero, Speakers, Pricing, Sponsors, Venue, FAQ, and Footer.

### Prompt 5
Help me organize my project structure, including file naming, folder structure, GitHub repository setup, and README documentation.

### Prompt 6
Guide me in debugging HTML and CSS issues, explain why errors occur, and recommend best practices to write cleaner and more maintainable code.

## Declaration
ChatGPT was used as a learning assistant for guidance, explanations, debugging, and design suggestions throughout the project. The project was developed, organized, tested, and finalized by the author.
